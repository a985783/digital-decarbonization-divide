## 1. Institutional Review Board (IRB) Requirements

### 1.1 IRB Submission Checklist

#### Pre-Submission Requirements

- [ ] **IRB Determination Request**
  - [ ] Submit project summary to IRB for determination of review level
  - [ ] Receive determination: Exempt / Expedited / Full Board Review
  - [ ] Document IRB determination letter

- [ ] **Protocol Documentation**
  - [ ] Complete IRB application form
  - [ ] Attach full study protocol (see policy_experiment_design.md)
  - [ ] Include data collection instruments (surveys, consent forms)
  - [ ] Provide recruitment materials
  - [ ] Submit investigator CVs and training certificates

- [ ] **Informed Consent**
  - [ ] Draft informed consent forms for all participant types:
    - [ ] Household participants (local language)
    - [ ] Firm managers (local language)
    - [ ] Government officials
    - [ ] District administrators
  - [ ] Ensure consent forms include:
    - [ ] Study purpose and procedures
    - [ ] Risks and benefits
    - [ ] Voluntary participation statement
    - [ ] Right to withdraw
    - [ ] Data use and confidentiality
    - [ ] Contact information for questions
    - [ ] Compensation details (if applicable)

- [ ] **Risk Assessment**
  - [ ] Document minimal risk determination
  - [ ] Identify potential risks:
    - [ ] Privacy breaches
    - [ ] Economic harm from treatment assignment
    - [ ] Social/political sensitivities
    - [ ] Data security risks
  - [ ] Describe risk mitigation strategies

### 1.2 Ongoing IRB Obligations

- [ ] **Continuing Review**
  - [ ] Schedule annual continuing review
  - [ ] Submit progress reports
  - [ ] Report any protocol modifications
  - [ ] Document adverse events

- [ ] **Protocol Modifications**
  - [ ] Submit amendment requests for any changes:
    - [ ] Sample size adjustments
    - [ ] New data collection methods
    - [ ] Additional study sites
    - [ ] Changes to interventions
  - [ ] Await approval before implementing changes

- [ ] **Study Closure**
  - [ ] Submit final report within 90 days of completion
  - [ ] Archive all IRB documentation
  - [ ] Retain records for required period (typically 3+ years)

## 3. Data Protection and Privacy

### 3.1 Legal Compliance

- [ ] **Data Protection Laws**
  - [ ] Identify applicable data protection regulations:
    - [ ] GDPR (if EU citizens involved)
    - [ ] Local data protection laws
    - [ ] Sector-specific regulations (energy, telecom)
  - [ ] Conduct Data Protection Impact Assessment (DPIA)
  - [ ] Appoint Data Protection Officer (if required)
  - [ ] Register with data protection authorities (if required)

- [ ] **Cross-Border Data Transfer**
  - [ ] Identify data transfer requirements
  - [ ] Establish legal basis for transfer:
    - [ ] Adequacy decisions
    - [ ] Standard Contractual Clauses (SCCs)
    - [ ] Binding Corporate Rules (BCRs)
  - [ ] Document transfer mechanisms

### 3.2 Data Security Protocols

- [ ] **Data Collection Security**
  - [ ] Use encrypted data collection platforms
  - [ ] Implement secure authentication
  - [ ] Enable audit logging
  - [ ] Train staff on data security

- [ ] **Data Storage Security**
  - [ ] Encrypt data at rest (AES-256)
  - [ ] Use secure servers (ISO 27001 certified)
  - [ ] Implement access controls (role-based)
  - [ ] Regular security audits
  - [ ] Backup procedures (encrypted, off-site)

- [ ] **Data Transmission Security**
  - [ ] Use TLS 1.3 for all transmissions
  - [ ] VPN for remote access
  - [ ] Secure file transfer protocols

- [ ] **Data Retention and Disposal**
  - [ ] Define retention periods:
    - [ ] Raw data: 7 years
    - [ ] Anonymized data: Indefinite
    - [ ] Identifiable data: 5 years post-publication
  - [ ] Establish secure deletion procedures
  - [ ] Document data destruction

### 3.3 Participant Privacy

- [ ] **Anonymization Protocols**
  - [ ] Remove direct identifiers
  - [ ] Apply pseudonymization techniques
  - [ ] Aggregate sensitive variables

- [ ] **Re-identification Risk Assessment**
  - [ ] Assess risk for each dataset

## 5. Intervention Ethics

### 5.1 Equipoise and Justification

- [ ] **Clinical/Policy Equipoise**
  - [ ] Document genuine uncertainty about intervention effectiveness
  - [ ] Justify withholding treatment from control group
  - [ ] Commit to delayed treatment for controls (Month 30)
  - [ ] Establish criteria for early stopping if clear benefit/harm

- [ ] **Beneficence**
  - [ ] Ensure interventions have potential for benefit
  - [ ] Design interventions based on best available evidence
  - [ ] Include co-benefits (energy savings, productivity)
  - [ ] Monitor for unintended negative consequences

### 5.2 Fair Participant Selection

- [ ] **Inclusion/Exclusion Criteria**
  - [ ] Justify criteria scientifically
  - [ ] Avoid unfair exclusion of vulnerable groups
  - [ ] Document rationale for district selection
  - [ ] Ensure equitable distribution of benefits/risks

- [ ] **Randomization Ethics**
  - [ ] Use fair randomization procedures
  - [ ] Prevent manipulation of assignment
  - [ ] Maintain allocation concealment
  - [ ] Document randomization integrity

### 5.3 Monitoring for Harm

- [ ] **Data Safety Monitoring**
  - [ ] Establish Data Safety Monitoring Board (DSMB)
  - [ ] Define stopping rules for harm
  - [ ] Schedule interim analyses
  - [ ] Report adverse events within 24 hours

- [ ] **Adverse Event Reporting**
  - [ ] Define reportable events:
    - [ ] Data breaches
    - [ ] Technology failures causing harm
    - [ ] Social conflicts related to treatment
    - [ ] Economic losses
  - [ ] Establish reporting procedures
  - [ ] Create response protocols

## 10. Emergency Protocols

### 10.1 Data Breach Response

- [ ] **Immediate Actions (0-24 hours)**
  - [ ] Contain the breach
  - [ ] Assess scope and impact
  - [ ] Notify data protection authorities
  - [ ] Notify affected individuals (if required)

- [ ] **Follow-up Actions**
  - [ ] Investigate cause
  - [ ] Implement corrective measures
  - [ ] Document incident
  - [ ] Review and update protocols

### 10.2 Serious Adverse Events

- [ ] **Response Protocol**
  - [ ] Immediate participant support
  - [ ] Medical attention if needed
  - [ ] Report to IRB within 24 hours
  - [ ] Investigate and document
  - [ ] Implement preventive measures
