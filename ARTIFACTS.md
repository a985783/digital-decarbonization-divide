# Replication Artifacts Map

Language / 语言: [English](ARTIFACTS.md) | [中文](ARTIFACTS.zh-CN.md)

## Core Datasets

- `data/clean_data_v4_imputed.csv`: baseline panel used by most phase scripts.
- `data/clean_data_v5_enhanced.csv`: enhanced feature version used by dashboard/comparison modules.

## Main Quantitative Outputs

- `results/causal_forest_cate.csv`: CATE estimates from `scripts/phase2_causal_forest.py`.
- `results/iv_analysis_results.csv`: IV estimates from `scripts/phase4_iv_analysis.py`.
- `results/placebo_results.csv`: placebo diagnostics from `scripts/phase4_placebo.py`.
- `results/mechanism_analysis_results.csv`: mechanism decomposition from `scripts/phase5_mechanism.py`.
- `results/external_validity_results.csv`: external validity checks from `scripts/phase6_external_validity.py`.
- `results/dynamic_effects.csv`: dynamic treatment effects from `scripts/phase7_dynamic_effects.py`.
- `results/sensitivity_analysis.csv`: Oster sensitivity outputs from `scripts/oster_sensitivity.py`.
- `results/dragonnet_comparison.csv`: model comparison outputs from `scripts/dragonnet_comparison.py`.

## Figures and Paper

- `results/figures/`: generated visualizations used in the manuscript.
- `paper.pdf`, `paper_cn.pdf`: compiled manuscript outputs from `compile_paper.sh`.

## Verification Reports

- `results/academic_consistency_guard_report.md`: generated by `scripts/academic_consistency_guard.py`.
- Preflight status from `scripts/preflight_release_check.py`.
